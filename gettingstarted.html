Download

The Maespa model is free without restrictions. Please click the link below to start the download. The zip file includes the executable (Windows only), and two sets of example files.

How to get started

Download the zip file from the link above, unzip it and place everything in one folder. Then, simply run the executable ("Maespa.exe") to produce several output files. All input files and output files are text only, there is no GUI.

Next, read the technical manual to modify the input files. There is also an R package available which makes it convenient to run the model, modify input files, and process output files.
